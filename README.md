# Poker - テキサスホールデム

> ⚠️ **α版** - 前提: [EmeraldBank](https://github.com/henntekosennsi1-rgb/EmeraldBank)（必須）

テキサスホールデムポーカーです。

## インストール

1. [EmeraldBank](https://github.com/henntekosennsi1-rgb/EmeraldBank/releases) をダウンロード
2. Poker をダウンロード
3. 両方を plugins フォルダに配置

## ゲームフロー

1. プリフロップ（2枚配布）
2. フロップ（3枚公開）
3. ターン（4枚目）
4. リバー（5枚目）
5. ショーダウン

## 役（強い順）

1. ロイヤルストレートフラッシュ
2. ストレートフラッシュ
3. フォーカード
4. フルハウス
5. フラッシュ
6. ストレート
7. スリーカード
8. ツーペア
9. ワンペア
10. ハイカード

## コマンド

| コマンド | 説明 |
|---------|------|
| `/poker join` | 参加 |
| `/poker leave` | 離脱 |
| `/poker bet <額>` | SB設定 |
| `/poker sign1~5` | 看板設定 |
| `/poker start` | 開始 |
| `/poker stop` | 停止 |

## セットアップ

1. 5つの看板を設置
2. 各看板を見て `/poker sign1~5`
3. `/poker bet 10` でSB設定
4. プレイヤーが `/poker join`
5. `/poker start` で開始
